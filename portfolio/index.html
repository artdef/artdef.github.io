小古華遷徙記事主題特展

Friday, Feb 12, 2021

Project Introduction 展名｜Vusam a kemasi Kuabar—從古華到永康，當代排灣族人的遷徙記事主題特展 本展為「Íng-khong•Kuabar─從古華到永康，當代排灣族人的遷徙記事計畫」的第二年成果，將為期兩年臺南永康小古華聚落的田野調查成果、遷徙史發展，搭配紀錄片播映，於臺南市原住民文物館進行展出。 展序｜ 臺灣島上原先以南島語族的原住民人口為主，直至16世紀，來自世界各地的不同族群，如荷蘭人、西班牙人、日本人、中國沿海地區的漢人、戰後遷臺的外省籍移民、甚至是當代的東南亞移工與婚姻移民，因著各種原因先後踏上這塊土地，歷經400多年的時代更迭與政權移轉變動，逐漸在臺灣構成多族群的移民社會，即使到了現在，「移動」仍在這座島嶼上持續發生。 1950年代後，政府開始推動輕工業發展，臺灣各地村落人口為尋求工作漸移往都市，臺南也成為移居的都市之一。1972年莉泰颱風所帶來的天災遭受迫害，屏東縣春日鄉古華部落之排灣族族人，為了重建原鄉部落，在地緣與血緣間的連結下，相繼移居至臺南永康，並逐漸形成有別於原鄉之聚落（以下簡稱：小古華聚落）。 本展透由這群小古華聚落之族人在過去半個世紀以來，於原鄉古華與臺南永康間所經歷的遷徙、凝聚、信仰與世代傳衍之過程，帶領民眾一同窺探蘊藏在臺南這塊土地，一段獨特的原住民族遷徙歷程。即便離鄉近50年，第一代移居的族人仍積極在新環境中，實踐出根源於原鄉的信仰、文化與價值觀，並悉心在信仰與文化傳承上耕耘，將此種子扎根、持續繁衍下去。 「Vusam a kemasi Kuabar」意為「來自古華的種子」。「Vusam」一詞在排灣族原意為小米種子，是用作隔年播種的上好種子，排灣族也相傳：「一對夫妻結合以後，第一個看到太陽的孩子，就是繼承人，就是Vusam。」Vusam亦指排灣族文化中的長嗣，長嗣除了要繼承家業，也承擔起整個家庭的責任，身為部落王族的Vusam更是維繫部落興盛的命脈。而這段古華部落族人遷徙永康的歷程中，我們也從族人身上，看見Vusam的核心價值再次體現。 Content Curation : 蔡立德 Exhibition Design : 劉培弘 Printout : 品羚廣告企業社 Exhibition Layout : 蔡立德、劉培弘、洪百加 Cultural Relics Provided : 小古華族人、福山門教會

2018臺灣文化日巡迴影展

Friday, Nov 16, 2018

Project Introduction 本案為2018年工作室成立前所承辦的第一個計畫，與南藝大藝史系的友人一同策劃辦理，具有藝術史研究背景的友人以「複音」概念作為主題，重新詮釋和包裝本次臺灣文化日影展，並依照所挑選的片單規劃6場具備議題討論的映後座談。 影展主題｜複音 × 再歌啟蒙思想： 2018 臺灣文化日影展 回到問題的現場 1921年10月17日，臺灣民主與文化運動先驅蔣渭水結合有識之士，創辦「台灣文化協會」，推動「文化啟蒙運動」，希冀提高臺灣人民的文化意識，進一步喚醒民族意識，推動以台灣為主體的價值觀。 如今我們身處21世紀，台灣社會科技發達、資訊傳播快速。當我們逐漸積累、沈澱出自身文化底蘊的同時，又如何理解何謂「文化」？或許回到帝國殖民統治、戰後光復、白色恐怖與現代化與全球化衝擊下的每個當局，透過影片，我們反覆質問「為什麼需要啟蒙」，以及「為何而啟蒙」？ 無論影片的導演採取紀錄片或劇情片的方式，對歷史重新提問或是問題化，都讓我們在觀影的同時，再次探問「文化」與「文化啟蒙」之於個人與群體的意義。本次影展延伸上一屆「啟蒙進行曲」的概念，以「複音x 再歌啟蒙思想」為題，希望以今日的角度，重新觀看「臺灣文化啟蒙」，進一步的以複數視野來打破「文化」與「文化啟蒙」的單一詮釋框架。 聆聽複數歷史的聲音 「複音」／「複音音樂」（polyphony）乃早期西方音樂系統衍生出來的一種音樂創演。通常指樂曲中含有兩條以上的獨立旋律，通過技術性處理，和諧地結合在一起。不同於主音音樂（homophony）由某一個聲部來支配、控制著其他的聲部；複音音樂中的每一個聲部皆是獨立的，往往難以確定哪一個才是最主要的聲部。 若20世紀初蔣渭水發起的「文化啟蒙運動」可視為一首「啟蒙進行曲」，將近一個世紀後的我們，或許可以「複音」的形式，為「文化啟蒙運動」重新編曲、演奏與賞析。 臺灣這塊土地經歷了許多政權的更迭，使得臺灣的歷史和文化脈絡相對複雜。因為這般複雜的歷史脈絡，臺灣難以明確地形構出自己的文化主體性。或者說臺灣的文化主體，須藉由多元主體層層推疊架構完成。由此，擺脫單一視點的觀看方法，不僅可能打破我們對於歷史與文化的既定認知與想像。由當代的紀錄片觀點或劇情片編寫，不斷回訪與探問「文化啟蒙」及「文化」。依此建構出複數歷史的觀點，是本次影展的宗旨。 Activity Execution : 蔡立德、施友傑 Visual Design : 劉培弘 Activity Host : 陳碩儀、張美陵、黃建龍等 Talker : 李承機、廖振富、黃胤毓等
