臺中北區家福中心牆面設計

2021-01-16| Category: 設計| website:

Project Introduction

本案為臺中市政府社會局家庭福利服務中心資訊牆面重新設計，結合該單位所服務區域之地理範圍、服務人口統計資料以及相關合作機構，規劃一面結合地圖，用來呈現上述資訊的磁鐵牆面，其數字及單位icon皆為磁鐵方便使用者更換。

Design : 劉培弘

Construction : 帆鳳室內裝修工程行

Printout : 品羚廣告企業社

Clients
臺中市社會局
Date
2020/12 - 2021/01
Location
臺中市北區家庭福利服務中心
