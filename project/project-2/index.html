行動教育箱製作

以透明魚為主題的教案、教具設計與開發

Project Details

  • Client : 國立海洋生物博物館
  • What We Did : 行動展示教具箱
  • Completed on : 2020/06 - 2020/10
  • Teaching Plan : 蔡立德、劉培弘
  • Visual Design : 劉培弘
  • Teaching Aid Production : 立德照相製版、太空森林
  • Teaching Aid Box : HS Rack Cases

Project Introduction


教具箱主題｜魚仔看透透—透明魚教具箱

本案與國立海洋生物博物館合作，以充滿特色的「透明魚」作為主題，開發設計一系列以國、高中生為對象的教案教材，教案分成生物及藝術兩類，其中包含1個核心教案及4個延伸教案，共5個教案提供教學者進行操作使用；除此之外，本次教具箱設計結合行動展示功能，以瑞克箱作為箱體，兼顧美觀與堅固功能，搭配內部教具擺放的客製化分層設計，讓使用者能夠更輕易取用及收納教具。
