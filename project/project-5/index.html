小古華紀錄片放映會

Vusam a kemasi Kuabar 從古華到永康—當代排灣族人的遷徙記事

Project Details

  • Client : 臺南市政府原民處
  • What We Did : 紀錄片放映座談會、音樂會
  • Completed on : 2019/10 - 2019/11
  • Location : 臺南永康區、屏東春日鄉
  • Activity Execution : 蔡立德
  • Stage Equipment : 王志皓
  • Activity Host : DJ尚恩、周華南、邱宗怡
  • Performer : 排灣女聲、段賽英
  • Talker : 小古華族人（紀錄片主角）
  • Staff & Photography : 施友傑、李亞勳、江茂源

Project Introduction


主題｜nia cinalivatan anga kuabar a tjalju yungkang—「大成小古華」當代排灣族人府城遷徙記事紀錄片系列放映會

「Vusam a kemasi Kuabar 從古華到永康—當代排灣族人的遷徙記事」的第一年計畫成果，將7支不同主題的紀錄片分別在臺南永康與屏東春日古華部落規劃了4場放映會，放映會中除了觀賞紀錄片外，也在其中2場次邀請來自屏東春日古華在地的「排灣女聲」現場演唱，而紀錄片主角段賽英牧師的好歌喉，也受邀在古華場次獻聲，而在影片播映之後，活動也邀請紀錄片主角現身分享他們所走過得這段半世紀遷徙故事。
