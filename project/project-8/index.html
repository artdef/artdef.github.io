Project Introduction


書名｜漆詣─南投縣漆藝發展史

本書設計為提案階段，設計師在設計上，以漆藝帶給人細緻、典雅的感受為意象；字體上，以流線造型與細緻的線條設計，讓書整體的視覺感，呼應漆的液體流動感，以及漆器裝飾的描金工藝質感。。
