小古華紀錄片拍攝與剪輯

Íng-khong•Kuabar─從古華到永康，當代排灣族人的遷徙記事計畫

Project Details

  • Client : 臺南市政府原民會
  • What We Did : 紀錄片拍攝與剪輯製作
  • Completed on : 2019/09 - 2020/12
  • Documentary Director : 邱宗怡
  • Documentary Editor : 陳惠萍
  • Director of Phase II : 洪辰嫣
  • Editor of Phase II : 洪辰嫣、馮鵬安

Project Introduction

2019年9月，工作室與紀錄片導演宗怡、攝影剪輯哈比合作拍攝小古華族人的遷徙史，在第一年由兩人完成7部不同主題，每部約10-15分鐘的短版紀錄片；第二年則因應機關要求，與導演辰嫣與剪輯鵬安合作，將原有的影片素材重新撰寫腳本並剪輯成長版紀錄片，紀錄片內容收錄於由臺南市政府原民會出版的《從古華到永康: 當代排灣族人的遷徙記事影音專刊》。
