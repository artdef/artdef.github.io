2020臺灣文博會宜蘭縣主題展區展示規劃設計

Project Details

  • Client : 宜蘭縣政府文化局
  • What We Did : 展示內容規劃與設計
  • Completed on : 2020/02
  • Exhibition Content Planning : 蔡立德、劉培弘
  • Exhibition Design : 劉培弘、王靖文

Project Introduction

本案為工作室於2020年1月參與「宜蘭縣政府文化局：2020臺灣文博會宜蘭縣主題展區規劃」之提案，雖然最後並沒有順利得標，但也將提案內容進行整理呈現。


策展主題與論述｜YMT宜蘭標準時間

故事從這裡開始。

總面積約2,143平方公里

總人口 約45.43萬人

全年雨日超過200天

縣花：國蘭

縣樹：臺灣欒樹

縣鳥：彩鷸

宜蘭的高山面迎東北季風，造就宜蘭雨水豐沛的氣候與水文特性，9至12月是颱風季節雨量最多，加上夏季對流雨旺盛，全年雨日超過200天。在宜蘭生活的人們早已學會與雨水共存，那是天賜的甘霖，滋養萬物山林，人們在宜蘭時區裡與土地共生共息，也摸索出與氣候環境共處的態度，放慢腳步，隨遇而安，成為宜蘭的慢活之道。本展之策展論述以國際標準時間GMT之概念發想，設定「YMT 宜蘭標準時間」作為展題，YMT意指Yilan Mean Time，用以表示在宜蘭這塊土地上，一群人生活所共同營造的時間感，小至個人，因此展區敘事框架以「時間感」作為串連的主體架構，並以宜蘭的多雨為開端，從山林到平原，進入人們的生活環境，藉以呼應宜蘭地理人文特色，同時展示相關原物料或雛型，再帶入其中的相關產業與品牌，為此，本展從以下三個面向切入提供策展觀點及敘事路徑:

  1. 以「YMT宜蘭標準時間」為策展主軸；

  2. 以「雨、霧／木、光／土、水」為策展核心；

  3. 從宜蘭的「時間感」與「自然意象」為概念出發。
