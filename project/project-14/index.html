展示規劃與設計提案

故宮國寶遊彰化展覽設計規劃

Project Details

  • Client : 彰化縣文化局
  • What We Did : 展示內容規劃與設計
  • Completed on : 2019/11
  • Exhibition Content Planning : 蔡立德、李亞勳、林君玲
  • Exhibition Design : 劉培弘
  • Illustration Design : 傅馨逸

Project Introduction

本案為工作室於2019年11月參與「彰化縣文化局：故宮國寶遊彰化展覽設計規劃」之提案，雖然最後並沒有順利得標，但也將提案內容進行整理呈現。


策展主題｜故宮國寶微服巡遊

透過此案將國寶文物結合在地特色，彰顯彰化「傳統產業」、「人文藝術」與「庶民生活」與國寶間之連結。


展期主題規劃｜

展示文物 ｜ 主題規劃
肉形石 ｜ 肉形石與彰化爌肉的24小時約會
黃玉鴨、白玉錦荔枝 ｜ 黃玉鴨與白玉錦荔枝的農事生活
黃玉髓三連章、玉鳳柄洗 ｜ 黃玉髓三連章與玉鳳炳洗的書畫課


主視覺設計｜

1.以故宮文物的尊榮意象作為設計概念，並以放大、仿金的字體來展現故宮文物的氣勢。


2.將故宮國寶結合活潑的色彩及人物角色造型，營造出國寶巡遊彰化充滿歡樂與驚喜的氛圍。


主題展區展示手法｜

配合故宮文物展示規劃，將國寶遊彰化分三檔三系列展出，內容包含第一檔主題/肉形石、第二檔主題/黃玉鴨與白玉錦荔枝，以及第三檔主題/黃玉髓三連章與玉鳳柄洗，此區展示設計採主題取向，著重在展覽概念與訊息的傳達、詮釋，輔以展示輔助工具
