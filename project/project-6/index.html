小古華記者會

Vusam a kemasi Kuabar 從古華到永康—當代排灣族人的遷徙記事

Project Details

  • Client : 臺南市政府原民處
  • What We Did : 記者會
  • Completed on : 2019/11、2020/11
  • Activity Execution : 蔡立德
  • Activity Host : DJ尚恩、DJ小喬
  • Staff : 施友傑、楊展華、黃瀞瑩

Project Introduction

「Vusam a kemasi Kuabar 從古華到永康—當代排灣族人的遷徙記事」計畫分別在第一年和第二年舉辦紀錄片與影音專刊發布記者會，並在記者會中播放短片，以及邀請小古華族人現場分享參與本次計畫過程的心路歷程，最後與現場的長官、貴賓合影留念。
