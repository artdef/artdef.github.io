久堂廠採訪與撰文

永豐餘久堂七十週年專刊

Project Details

  • Client : 永豐餘
  • What We Did : 資料研究整理、採訪撰文
  • Completed on : 2020/12 - 2021/04
  • Interview : 蔡立德
  • Writing : 蔡立德、林君玲

Project Introduction

位於高雄大樹的永豐餘久堂廠成立至今已邁向70年，總公司特別製作久堂七十週年專刊，並委由工作室負責採訪、研究與撰寫久堂廠的地方歷史風情、紙張發展故事、久堂員工採訪、手抄紙文化社與地方公益、大樹國小的鳳纖紙特色課程與久堂廠的連結等。
