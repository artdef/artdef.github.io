Project Introduction


展名｜Vusam a kemasi Kuabar—從古華到永康，當代排灣族人的遷徙記事主題特展

本展為「Íng-khong•Kuabar─從古華到永康，當代排灣族人的遷徙記事計畫」的第二年成果，將為期兩年臺南永康小古華聚落的田野調查成果、遷徙史發展，搭配紀錄片播映，於臺南市原住民文物館進行展出。


展序｜

臺灣島上原先以南島語族的原住民人口為主，直至16世紀，來自世界各地的不同族群，如荷蘭人、西班牙人、日本人、中國沿海地區的漢人、戰後遷臺的外省籍移民、甚至是當代的東南亞移工與婚姻移民，因著各種原因先後踏上這塊土地，歷經400多年的時代更迭與政權移轉變動，逐漸在臺灣構成多族群的移民社會，即使到了現在，「移動」仍在這座島嶼上持續發生。

1950年代後，政府開始推動輕工業發展，臺灣各地村落人口為尋求工作漸移往都市，臺南也成為移居的都市之一。1972年莉泰颱風所帶來的天災遭受迫害，屏東縣春日鄉古華部落之排灣族族人，為了重建原鄉部落，在地緣與血緣間的連結下，相繼移居至臺南永康，並逐漸形成有別於原鄉之聚落（以下簡稱：小古華聚落）。

本展透由這群小古華聚落之族人在過去半個世紀以來，於原鄉古華與臺南永康間所經歷的遷徙、凝聚、信仰與世代傳衍之過程，帶領民眾一同窺探蘊藏在臺南這塊土地，一段獨特的原住民族遷徙歷程。即便離鄉近50年，第一代移居的族人仍積極在新環境中，實踐出根源於原鄉的信仰、文化與價值觀，並悉心在信仰與文化傳承上耕耘，將此種子扎根、持續繁衍下去。

「Vusam a kemasi Kuabar」意為「來自古華的種子」。「Vusam」一詞在排灣族原意為小米種子，是用作隔年播種的上好種子，排灣族也相傳：「一對夫妻結合以後，第一個看到太陽的孩子，就是繼承人，就是Vusam。」Vusam亦指排灣族文化中的長嗣，長嗣除了要繼承家業，也承擔起整個家庭的責任，身為部落王族的Vusam更是維繫部落興盛的命脈。而這段古華部落族人遷徙永康的歷程中，我們也從族人身上，看見Vusam的核心價值再次體現。
